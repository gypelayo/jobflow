Install JobFlow

Installation takes about 2 minutes

1

Download the extension

Click the button below to get the latest Chrome version:

Download for Chrome
2

Unzip the file

Extract the ZIP to a folder. Remember the location.

3

Open extensions page

Go to chrome://extensions/ in Chrome, or click the three-dot menu → Extensions → Manage Extensions

4

Enable Developer mode

Toggle the "Developer mode" switch in the top-right corner.

5

Load the extension

Click "Load unpacked" and select the unzipped folder.

6

You're done

The JobFlow icon should appear in your toolbar. Click it to start tracking.

Temporary installation

Firefox temporary add-ons are removed when you close the browser. We're working on official Firefox Add-on store approval.

1

Download the extension

Click the button below to get the latest Firefox version:

Download for Firefox
2

Unzip the file

Extract the ZIP to a folder on your computer.

3

Open Firefox debugging

Go to about:debugging#/runtime/this-firefox in Firefox

4

Load temporary add-on

Click "Load Temporary Add-on" and select any file from the unzipped folder (e.g., manifest.json).

5

Extension loaded

JobFlow appears in your toolbar. Note: you'll need to repeat this after restarting Firefox.

Troubleshooting

Extension icon doesn't appear
Check that the extension is enabled in chrome://extensions/ or about:addons. You may need to pin it to the toolbar.
"Load unpacked" is grayed out
Make sure "Developer mode" is enabled in the top-right corner of the Extensions page.
"No LLM provider configured"
Click the JobFlow icon → Settings and configure an AI provider, or install Ollama for local processing.
Firefox extension disappears after restart
This is expected with temporary add-ons. We're working on Firefox Add-on store approval.
